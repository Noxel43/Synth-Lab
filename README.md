# Synth Lab features:
- Polyphonic Synth
- Virtual Theremin
- Beat Machine
- Galaxy Sequencer
- Arpeggiator
- Sampler
- Chord pads
- Noise box
- Drone
- Bubble synth
- XY Pad

- Exporting
- Recording and Playback

All of them are pretty simple in functionality, however, this is definetly not for creating proffesional music, it's for having fun.
